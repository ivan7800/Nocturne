# Sample pack opcional

Esta carpeta existe para que el usuario pueda añadir sus propios WAV/MP3/OGG libres de derechos.

Nocturne funciona sin samples, pero en modo híbrido alcanza calidad superior cuando encuentra archivos nombrados como el ID del sonido:

- lluvia-pesada.wav
- fuego-chimenea.wav
- viento-bosque.wav
- pasos-madera.wav
- tormenta-lejana.wav

Los archivos de audio reales no se incluyen para evitar problemas de copyright y mantener el repositorio ligero.
