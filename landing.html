Dependencias runtime

Para narradores exigentes

🎚
Capas independientes
Hasta 12 capas simultáneas. Cada una con volumen, ritmo y nivel de corrupción propio. Las capas se degradan hacia su equivalente perturbador.
🧭
Eje emocional
Plano 2D interactivo (calma↔tensión / placer↔terror). Arrastra un punto y Nocturne construye la mezcla automáticamente por intención narrativa.
🎬
Modo director
Automatización temporal con arcos dramáticos: tensión creciente, desvanecimiento, oleaje o caos impredecible. La escena evoluciona sola.
💀
Lovecraft & ocultismo
14 sonidos de la mitología lovecraftiana, 7 de sectas y rituales, frecuencias psicoacústicas reales (19 Hz, 111 Hz, 528 Hz, Schumann 7.83 Hz).
🔬
Motor realista + samples
Por defecto genera audio con Web Audio API. Si quieres realismo grabado, puedes cargar tus propios WAV/MP3 locales nombrados como cada ID de sonido.
🧘
Frecuencias de placer
Cuencos tibetanos en afinación justa, ondas binaurales alpha/theta/delta, Nada Brahma a 136.1 Hz, canto de ballenas, didgeridoo. El mismo rigor que el terror.
🌐
Compartir por URL
La escena se codifica en el propio enlace. Sin servidor, sin base de datos. Un link = una mezcla completa lista para compartir con cualquiera.
📴
Funciona offline
PWA instalable. Sin dependencias runtime, sin llamadas de red, sin seguimiento. Abre el archivo o despliega en GitHub Pages y funciona.

Instalación

No requiere backend. Descomprime el ZIP, entra dentro de la carpeta generada y sube esos archivos a la raíz del repositorio. No subas la carpeta contenedora.

# Prueba local recomendada
python3 -m http.server 8000

# Abre en el navegador
http://localhost:8000/index.html

# GitHub Pages
Settings → Pages → Deploy from branch → main → /root

Compatible con Chrome, Firefox, Safari y Edge modernos.
Para efectos binaurales completos, usa auriculares.
